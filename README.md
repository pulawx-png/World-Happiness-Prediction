# World Happiness Prediction Analysis

## Project Overview
This project explores the 2023 World Happiness Report to identify the primary drivers of national well-being. Using Machine Learning, I predicted happiness scores based on economic, social, and health factors.

## Key Findings
* **Top Predictors:** Logged GDP per capita, Social Support, and Healthy Life Expectancy showed the strongest correlations with happiness.
* **Best Model:** Linear Regression and Random Forest models were evaluated, with Linear Regression achieving an R² score of approximately 0.87.

## Technologies Used
* Python (Pandas, Scikit-learn, Seaborn)
* Jupyter Notebook
* Machine Learning (Linear Regression, Random Forest, Gradient Boosting)
